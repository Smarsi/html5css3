Requisitos do Sistema para rodar o jogo

Windows S.O.: Windows XP, Vista, 7 (ou mais avançado); Memória: 1GB Windows XP, 2GB Vista/7; Placa Gráfica: Shader 3.0 ou melhor, DirectX 9 compatível com 256 MB Nvidia GeForce 8000 séries / ATI Radeon 3000 séries; Placa Som: DirectX 9.0C ou recente; Espaço livre: 7.6 GB de HD livre.


Mac S.O.: OS X 10.6.6 (ou mais avançado); Processador: Intel Core 2 Duo / AMD equivalente (2GHz ou melhor); Memória: 2GB RAM; Placa Gráfica: Nvidia GeForce 8600M (ou melhor) / ATI Radeon HD 2400 (ou melhor); Espaço livre: 7.6 GB de HD livre.
